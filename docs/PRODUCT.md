# Product Philosophy

## Scoring philosophy

SayReady scoring is rubric-bound and coaching-oriented:

- Scores reflect **observable communication outcomes** in a specific scenario (clarity, task completion, organization, politeness, and repair strategies).
- Evaluations prioritize **actionable feedback** over absolute ranking.
- Category feedback should map to stable mistake keys so users can track repeated issues over time.
- Numerical scores are constrained to **0–100** and interpreted as directional progress, not a standardized proficiency certification.
- Prompt behavior is versioned (`EVAL_PROMPT_VERSION`) so historical evaluations can be interpreted in context.

## Non-goals

- SayReady does **not** make claims of accent precision measurement or accent-based correctness scoring.
- SayReady does **not** attempt biometric speaker identification.
- SayReady does **not** replace formal language proficiency exams.
